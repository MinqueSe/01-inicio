¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML

Html se usa para encerrar partes del contenido para que se vean y comporten de otra forma

< head > < p > < meta > < body > < html > < h > < title > < !DOCTYPE >

¿Cómo es la anatomía de un elemento HTML? Dar un ejemplo con la etiqueta < p >

<p> es la etiqueta de apertura

</p> es la etiqueta cierre

en el centro va el texto

ejemplo: < p > Palta con Limon y sal </p>

¿Cuál es la anatomía básica del HTML? Escribir su código y definir <!DOCTYPE html> <html> </html> <head></head> <meta charset="utf-8"> <title></title> <body></body></h2>
